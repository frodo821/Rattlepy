rattlepy.utils.createHeader(metas=None, title=None)¶

Create head element. Usage:

with createHeader(
  [{"charset": "utf-8"}],
  "Page Title"):
    ...

This function equals to the code:

with head():
  for m in [{"charset": "utf-8"}]:
    meta(**m)
  setTitle("Page Title")
